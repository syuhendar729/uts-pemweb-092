Link deployment: https://uts-pemweb-092.vercel.app/

Aplikasi ini berfungsi untuk men-tracking aset cryptocurrency secara real-time. Fitur-fitur utama yang tersedia meliputi:
- **Daftar Cryptocurrency**: Menampilkan tabel berisi data harga, market cap, dan perubahan harga dalam 24 jam terakhir.
- **Tombol Refresh Data**: Memperbarui data harga secara langsung dengan indikator loading saat proses berlangsung.
- **Filter Berdasarkan Harga**: Memungkinkan pengguna untuk memfilter daftar coin berdasarkan rentang harga tertentu.
- **Detail Cryptocurrency**: Menampilkan informasi detail setiap coin, termasuk grafik pergerakan harga menggunakan library chart.
- **Kalkulator Portofolio**: Pengguna dapat memasukkan jumlah coin yang dimiliki untuk menghitung total nilai asetnya secara otomatis.
